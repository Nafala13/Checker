How it works

1. Load your signal log

Reads signal_log.xlsx from Intraday Radar Bot. Filters signals by minimum score (default: 6/9). Removes duplicate signals using the smart 30-minute time window dedup per coin.

2. Fetch candle data from Binance

For each unique signal, pulls historical 15m candles from Binance API starting from the signal's timestamp. Data is cached locally — Binance is only called once per signal, even across multiple windows.

3. Scan candle-by-candle for TP / SL hits

Iterates through each candle after the signal. Checks if the high or low touched the TP or SL level. Records exactly which candle it hit and at what time. For signals that didn't hit, compares current live price from Binance.

4. Output to Excel

Generates a color-coded 5-sheet Excel with results per window plus a summary comparison table with visual win rate bars.

Result Statuses

Every signal gets one of these statuses per window:

TP ✅ — Take profit hit SL ❌ — Stop loss hit ALMOST TP 🟡 — 70%+ toward TP ALMOST SL 🟠 — 70%+ toward SL OPEN ⏳ — Not hit within window
Smart Deduplication — Opsi B

30-minute time window per symbol

15:31  ARUSDT  6/9  → KEEP   (first appearance)
15:40  ARUSDT  7/9  → SKIP   (9 min from 15:31 — duplicate)
15:48  ARUSDT  7/9  → SKIP   (17 min from 15:31 — duplicate)
16:05  ARUSDT  6/9  → KEEP   (34 min from 15:31 — NEW setup!)
16:18  ARUSDT  6/9  → SKIP   (13 min from 16:05 — duplicate)

The same coin appearing multiple times within 30 minutes is treated as one signal opportunity — only the highest-score instance is evaluated. This gives a fair, non-inflated win rate that reflects real trading decisions.

Excel Output — 5 Sheets

📊 Summary sheet (first tab)

Comparison table showing TP, SL, win rate, average profit %, and average candle to TP across all 4 windows. Includes a visual ASCII win rate bar chart and automatically highlights the best-performing window.

⏱ 1H / 2H / 5H / 24H sheets

Per-window detail sheets. Each row is color-coded by result. Header includes a live stats bar showing win rate, total TP/SL counts, and averages for that window. Columns: Datetime, Symbol, Trend, Score, Entry, SL, TP, Risk%, Result, Hit Candle, Hit Time, Hit Price, Current Price, P/L %, Note.

How to run

Place signal_result_checker.py and your signal_log.xlsx in the same folder, then run:
